# This is NOT the Next.js you know

This version has breaking changes — APIs, conventions, and file structure may all differ from your training data. Read the relevant guide in `node_modules/next/dist/docs/` before writing any code. Heed deprecation notices.


Project Overview

This project is a premium frontend-only investment website focused on gold and precious metals.

The product direction is:

luxury fintech
modern investment platform
cinematic premium UI
visually polished frontend experience

The project must feel modern, elegant, responsive, and production-ready.

Core Stack
Next.js 15+ App Router
TypeScript
Tailwind CSS
Framer Motion
shadcn/ui

Optional:

GSAP
Lenis smooth scrolling
Three.js / React Three Fiber
Main Priorities

Priority order:

Visual quality
Frontend polish
Smooth animations
Responsive behavior
Performance
Clean architecture

Business logic is secondary.
